## ✨ Features ✨

**Extended Beacon Range** is a lightweight quality-of-life mod that increases the effective range of Beacons by **x3.0**, while preserving vanilla balance and performance.

✨ Lightweight
<br />
✨ No performance impact
<br />
✨ No new mechanics or blocks
<br />
✨ No configuration required
<br />
✨ Just install and play

---

## 📏 Range comparison

The effect radius of every Beacon pyramid level is multiplied by **3x**.

| Pyramid Level     | 🟥 Vanilla | 🟪 Extended Beacon Range | 📐 Diameter |
| ----------------- | :--------: | :----------------------: | :---------: |
| **Level 1**       |     20     |          **60**          |     121     |
| **Level 2**       |     30     |          **90**          |     181     |
| **Level 3**       |     40     |         **120**          |     241     |
| **Level 4 (Max)** |     50     |         **150**          |     301     |

> Values shown represent the **effect radius in blocks**.
